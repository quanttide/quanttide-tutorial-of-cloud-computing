## 公有云 vs 私有云 vs 混合云

- 公有云：云厂商提供。
- 私有云：机子是自己的，系统是云厂商的。
- 混合云：公有云 + 私有云。

### IaaS PaaS SaaS

"aaS": as a service

Service: 云服务 cloud computing service

I Infra: 基础设施即服务
P Platform：平台即服务
S Software: 软件即服务

SaaS: License -> subscription
PaaS: "Platform"
IaaS: Infra
